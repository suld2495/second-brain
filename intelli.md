# 개인 비서 앱 MVP

## 핵심 컨셉

> **Mem.ai** = 기억 저장소 (나중에 찾으면 됨)
>
> **내 앱** = 비서 (알아서 챙겨줌)

---

## 핵심 개념

| 개념 | 설명 |
|------|------|
| **메모** | 사용자가 자유롭게 작성하는 텍스트 (이미지 포함 가능) |
| **일정** | 메모에서 추출된 날짜 + 할 일 조합 |
| **반복 일정** | 주기적으로 발생하는 일정 (매일, 매주 X요일, 매월 X일 등) |

---

## 사용자 시나리오

### 1. 메모 작성
- 사용자가 생각나는 대로 적음
- "내일 치과 예약", "A는 고양이 좋아함", "매주 금요일 운동" 등

### 2. 검색/질문
- "A 선물 뭐 사지?" → "A는 테니스 좋아함" 메모 찾아서 답변
- "이번 주 일정 알려줘" → 일정 목록 보여줌

### 3. 오늘 할 일 확인
- 앱 열면 오늘 해야 할 것들이 보임
- 반복 일정도 해당 요일에 자동으로 뜸

### 4. 일정 변경
- "운동 토요일로 옮겨줘" → 채팅으로 변경

---

## MVP 기능

### 1단계: 기본
- **메모 작성** - 자유 텍스트 저장
- **의미 단위 검색** - LLM reasoning 기반 (단순 시맨틱 검색 ❌)

### 2단계: 일정
- **일정 자동 추출** - 메모에서 날짜 감지 → 일정 등록
- **오늘 할 일 뷰** - 오늘 날짜 기준 일정 목록
- **채팅으로 일정 변경**

---

## MVP 우선순위

| 순서 | 기능 | 이유 |
|------|------|------|
| 1 | 메모 작성 | 기본 |
| 2 | 의미 단위 검색 | 핵심 차별점 |
| 3 | 일정 자동 추출 | Mem.ai와 차별화 |
| 4 | 오늘 할 일 뷰 | 일정 확인용 |
| 5 | 채팅 일정 변경 | 편의 기능 |

---

## 날짜 표현 (인식해야 할 것들)

| 표현 | 의미 |
|------|------|
| 오늘, 내일, 모레 | 상대 날짜 |
| 이번 주 금요일 | 상대 요일 |
| 1월 15일, 1/15 | 절대 날짜 |
| 매일, 매주 X요일, 매월 X일 | 반복 |
